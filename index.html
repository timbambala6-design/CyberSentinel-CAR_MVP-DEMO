Architecture Technique

Stack complet · MVP déployable en 3 semaines · Optimisé pour les infrastructures CAR

01
🤖 Moteur IA / NLP — Détection de fraude
Python · spaCy · Scikit-learn · Custom NLP Model
▼

Modèle NLP léger entraîné sur +5,000 SMS frauduleux documentés en Afrique francophone. Adapté au français camerounais, centrafricain et au sango. Score de risque 0–100 en moins de 500ms. Fonctionne sans connexion constante grâce au modèle embarqué.

spaCy 3.7 scikit-learn BERT multilingue FastAPI Redis cache Docker
02
💬 Interface WhatsApp Business
Twilio · WhatsApp Business API · Webhook architecture
▶

Bot WhatsApp accessible depuis n'importe quel smartphone avec WhatsApp. Flux conversationnel multilingue (FR / Sango). Intégration Twilio pour gestion des messages entrants et sortants. Boutons d'action rapide pour signalement et apprentissage. Transcription vocale pour utilisateurs non-alphabètes.

Twilio API WhatsApp Business Node.js Webhooks Multilingual NLP Voice transcription
03
📟 Passerelle USSD — Accès zéro-data
Africa's Talking USSD API · Orange Centrafrique
▶

Canal USSD (composition *XXX#) permettant à TOUT abonné Orange en RCA d'accéder aux services CyberSentinel sans données internet. Crucial pour les zones rurales et les utilisateurs de téléphones basiques. Signalement de fraude, leçon quotidienne et vérification de numéro disponibles en USSD.

Africa's Talking USSD Gateway Orange OMEA API Zero-data access
04
🗄️ Base de données de renseignement fraude
PostgreSQL · Python ETL · Automated Reporting
▶

Base de données communautaire des numéros frauduleux, messages phishing et profils d'arnaques. Données anonymisées et conformes RGPD. Pipeline ETL automatisé qui génère chaque semaine un rapport d'intelligence fraude pour les partenaires (Orange, régulateurs, ONG humanitaires). Premier référentiel de cybersécurité mobile de l'Afrique centrale.

PostgreSQL 16 Python ETL AWS Lightsail GDPR compliant Auto-reporting
05
📊 Dashboard partenaires
React · D3.js · REST API · Role-based access
▶

Interface web sécurisée pour Orange Centrafrique, COBAC (régulateur bancaire) et partenaires humanitaires. Visualisation temps réel des hotspots de fraude par quartier de Bangui, types d'arnaques, tendances hebdomadaires. Export automatique des rapports mensuels en PDF/CSV. Accès rôle par rôle (Orange, régulateur, ONG).

React 18 D3.js FastAPI REST JWT auth PDF export
